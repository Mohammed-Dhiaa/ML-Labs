# Lab 9 - Decision Trees and Random Forest

This folder contains the notebooks and datasets used for the Lab 9 machine learning exercises focused on Decision Trees and Random Forest models.

## Contents

- `01-Decision Trees and Random Forests.ipynb`: Core concepts and guided practice for Decision Trees and Random Forests.
- `02-Decision Trees and Random Forest Project.ipynb`: Project notebook applying the methods to real datasets.
- `loan_data.csv`: Loan dataset used for model training and evaluation.
- `kyphosis.csv`: Small classification dataset used for decision tree practice.

## Goal

Build and evaluate tree-based classification models, compare performance, and understand how ensemble methods such as Random Forest improve generalization.
